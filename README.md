# Trading Card Generator

## ⚡️ Quick Start

* Install dependencies
```bash
yarn install
# or
npm install
```

* Set up environment file at `.env.local` with:
```
CLOUDINARY_CLOUD_NAME="<Your Cloudinary Cloud Name>"
```

* Start development server
```
yarn dev
# or
npm run dev
```

* Check out your cards at http://localhost:3000/
